## To run the front-end application please refer to the commands below.
#### Navigate to the /buddi directory and install:
```
npm install
or 
yarn install
```
#### Then to start the app run:
```
npm run start 
or
yarn run start
```
## To run the back-end app please refer to the commands below.
#### Navigate to the /buddi-backend directory and install:
```
npm install 
or 
yarn install
```
#### Then to start the server run:
```
npm run start 
or
yarn run start
```
